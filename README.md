# 🗳️ Poll App

A simple and interactive Poll Application built using **HTML**, **Tailwind CSS**, and **JavaScript**, with **Local Storage** for data persistence. This app allows users to create polls, vote on polls, and view results — all without a backend.

---

## 🚀 Features

* 🔐 **User Authentication (Local Storage Based)**

  * Users can log in with a username
  * Session is stored locally in the browser

* 📝 **Create Polls**

  * Users can create custom polls with multiple options

* 📊 **View Poll Results**

  * See real-time vote counts for each option
  * Results update instantly after voting

* 👤 **User-Specific Poll Management**

  * Users can view all polls they have created
  * Track performance of their own polls

* 🌍 **Explore All Polls**

  * Users can see polls created by others
  * Participate by selecting their choice

* ✅ **Voting System**

  * Each user can vote on polls
  * Votes are saved in local storage

---

## 🛠️ Tech Stack

* **HTML** – Structure of the application
* **Tailwind CSS** – Styling and responsive design
* **JavaScript (Vanilla JS)** – Application logic
* **Local Storage** – Data persistence (users, polls, votes)

---

## 📌 Conclusion

This Poll App is a beginner-friendly project to understand how front-end technologies and browser storage can be used to build fully functional applications without a backend.
